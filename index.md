## üöß Setting Up the Bot

### Step 5: Create a Telegram Bot

1. Open Telegram.
2. Search for "BotFather" and start a chat.
3. Use the command `/newbot` and follow the prompts to create your bot.
4. Copy the token that BotFather gives you. You will need this token for your bot to work.

### Step 6: Configure Your Bot

1. Find the configuration file in the directory where you installed the bot.
2. Open the configuration file with a text editor.
3. Replace `YOUR_BOT_TOKEN` in the file with the token you copied from BotFather.

### Step 7: Run the Bot

1. Return to the terminal or command prompt.
2. Navigate to the bot directory.
3. Use the command `./your_bot_executable` (or equivalent) to start your bot.
